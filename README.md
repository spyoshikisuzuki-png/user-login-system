# 予約管理システム（PHP）

## 概要
顧客ログインと予約機能を備えたWebアプリです。

## 機能
- 会員登録・会員情報変更
- ログイン・ログアウト
- 予約登録・確認・完了画面
- 入力値チェック
- データベース連携（MySQL・PDO）

## 使用技術
- PHP 7.1.33
- MySQL
- Bootstrap 5
- JQuery UI

## 開発の工夫点
- セッションによる多画面データ保持
- SQLインジェクション対策（プリペアドステートメント）
- 予約番号の自動生成（連番管理）
